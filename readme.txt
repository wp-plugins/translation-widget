Lets the user choose language for the Translate plugin

== Description ==

This plugin adds a widget that lists all the installed languages for the Translate plugin,
with activation-links.

== Installation ==

1. Install the Translate plugin
2. Upload the `translate-plugin` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Set up the Translate plugin according to its instructions.
5. Go to the Widget screen and add the Language Widget to the list
6. Click Save Changes and check out your site!
